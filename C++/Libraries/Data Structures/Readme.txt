This library includes all data structures and are linked as a static library.
User can add this library to project and can freely continue using these data structures as ALL IN ONE PACKAGE.

Included Data Structures
1. Stack
2. Singly Linked List
3. Doubly Linked List
4. Queue
5. Abstract Data Type (ADT)
6. Sparse Matrix
7. AVL
8. BST
9. Min Heap
10. Max Heap
